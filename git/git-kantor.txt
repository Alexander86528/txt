3.3 Команда checkout при незакоммиченных изменениях.

с удалением незакоммиченных изменений:
1) git checkout --force master(переключает с какой-то ветки на ветку мастер)
git checkout -f master(короткий вариант команды выше).

2) git checkout -f HEAD (для удаления ненужных изменений смены ветки нет).
git chckout -f(короткий вариант команды выше).

с сохранением незакоммиченных изменений:
1) git stash(чтобы убрать изменения)
   git stash pop(чтобы вернуть изменения).
ветки не запоминает.
