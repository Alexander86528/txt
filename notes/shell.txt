    COMMAND(https://hackware.ru/?p=1851)
ctrl+A - переход на начало строки(также Home);
ctrl+U - удаление текущей строки;
ctrl+C - (q)прервать выполнение команды;
ctrl+r - реверсивный поиск.

    
Символ ";" для ввода нескольких команд в одной строке, или использование &&(логическое) если выполняется команда первая, то выполняется команда вторая.

pwd
print working directory

chmod +x <file>
change mode - смена режима

Команды chown и chgrp.
chown "владелец" "файл" - права владельца
chgrp "владелец" "файл" - права группы

Удаление директорий утилита rm
sudo rm -rf "file"

Копирование  cp
-r  рекурсивно
-p  c сохранением прав
