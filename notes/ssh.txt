1)установка ssh
sudo apt install openssh-server 

2) Статус ssh
sudo systemctl status ssh

3) Ubuntu поставляется с утилитой настройки брандмауэра под названием UFW. Если в вашей системе включен брандмауэр , обязательно откройте порт SSH:
sudo ufw allow ssh

4)Отключить ssh север
sudo systemctl disable --now ssh

5) Включить ssh сервер
sudo systemctl enable --now ssh
